# SITER Internship 2026 — Git Demo Project
## About This Project
This is a demonstration project created as part of Assignment #1 for the SITER Academy Summer Internship 2026 program (Norge).
## What I Learned
-	How to install and configure Git on Windows
-	How to create a GitHub account and public repository
-	How to write documentation using Markdown (README.md)
-	How to use core Git commands: init, add, commit, push, pull
-	Understanding branches and version control workflow
## Git Commands Used
git init # Initialize repository git add . # Stage all changes
git commit -m # Commit with a message git push # Upload to GitHub
## Tools Used
-	Git (Version Control)
-	GitHub (Repository Hosting & Portfolio)
-	Windows Command Prompt
